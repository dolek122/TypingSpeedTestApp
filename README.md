# Typing Speed Test
Aplikacja JavaFX do testowania szybkości pisania.  

## Funkcje  
- Wybór poziomu trudności (Łatwy, Średni, Trudny)  
- Wybór czasu trwania testu (15s, 30s, 60s)  
- Podświetlanie błędów w czasie rzeczywistym  
- Historia wyników na wykresie  
- Tryb jasny i ciemny

## Wymagania
- Java 11+
- JavaFx
